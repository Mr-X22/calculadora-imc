Calculadora de IMC
Aplicación web que permite calcular el Índice de Masa Corporal (IMC) a partir del peso y la estatura del usuario. El sistema muestra el resultado numérico y una clasificación basada en los rangos establecidos por la OMS.

Funcionalidades
* Ingreso de nombre, peso y estatura
* Cálculo automático del IMC mediante JavaScript
* Clasificación del resultado (bajo peso, normal, sobrepeso u obesidad)
* Validación de datos para evitar errores
* Indicador visual que muestra la categoría en una barra de colores

Tecnologías
* HTML: estructura de la página
* CSS: diseño e interfaz visual
* JavaScript: lógica del cálculo y validaciones

Uso
1. Ingresar nombre, peso (kg) y estatura (m)
2. Presionar el botón "Calcular"
3. Visualizar el IMC junto con su interpretación y posición en el indicador
